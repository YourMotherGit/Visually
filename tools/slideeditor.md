# Slide Editor 

To create your presentation and fully customizable slides, change the background color and personalize the content until it suits you and your presentation.

## Functions

- Text 
- Image
- Formula
- Shape
- Lottie
- List
- Table
- Code
- Icon
- Character
---

## - Text

Easily create text boxes into which any text can be written. This text can be customized to match the design in terms of color, font, and size.

## - Image

Upload an image or provide a link to one. This can then be placed anywhere on the foil and resized as desired.

## - Formula

A simple tool that makes it easier to write formulas or illustrate them in a presentation.

## - Shape

Choose from various shapes which you can use to highlight or decorate elements or your foil.

## - Lottie

Provide a link to a Lottie animation and you can move it like images on the slide and adjust its size.

## - List

Simplifies the presentation of lists, you can choose between bullet points and numbers, thus making it easier to present tables of contents or simple lists.

## - Table

Enables the simplified creation of small tables for presentations color, number of columns and rows are individually customizable.

## - Code

It allows short lines of code from various programming languages ​​to be shown in the presentation in order to explain relationships or meaning more easily.

## - Icon

An embedding of Lucide Icons, simply select and place on the slide, color and size are also customizable.

## - Character 

A character that can be easily programmed via element drag and drop and walks across the slide to highlight elements in the presentation.
